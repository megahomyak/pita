"I don't want to save" => remove the data and the flag
"I changed my mind, I do want to save" => restore the data and the flag
Entering => valid data => auth
Auth button => auth
Auth: valid data and checkbox? => save

States:
  No data saved
  Invalid data saved
  Valid data saved

Scenarios:
  User opens the app
    No data saved => Do nothing
    Invalid data saved => Fill the fields
    Valid data saved => Authenticate
  User edits the text field
    * => Save the data, drop the flag
  User presses the auth button
    No data saved => Save the data
  User turns the "Remember me" checkbox off
  User turns the "Remember me" checkbox on

-----

Remember me + successful login => saving the data
Toggling the checkbox off => removing the data
